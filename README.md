# 民航法規手機刷題工具 v4.1.1 

這是一個可直接部署到 GitHub Pages 的靜態刷題工具。v4.1.1 修正測試網址載入問題，並保留 v4.1 雙語顯示邏輯：英文原文為主、中文小字輔助，並保留手冊/文件縮寫原文。

## 檔案說明

- `index.html`：主程式與 182 題題庫
- `manifest.webmanifest`：手機加入主畫面設定
- `sw.js`：離線快取設定
- `icon-192.png` / `icon-512.png`：手機圖示

## 使用方式

部署到 GitHub Pages 後，使用者只需要用手機瀏覽器開啟網址即可刷題。

iPhone：用 Safari 開啟網址後，可點分享 → 加入主畫面。  
Android：用 Chrome 開啟網址後，可加入主畫面。

## 注意

第一次開啟需要網路。成功載入後，瀏覽器會快取檔案，之後可在同一台手機上離線使用。答題進度會保存在該瀏覽器的 localStorage。


## v4 更新
- 題目清單新增頂部與浮動「隱藏題目清單」按鈕，不需要滑到最下方即可關閉。
- 原題庫文字保留，英文題目與英文答案選項新增中文對照。
- 確認答案後，解析新增中文與 English explanation 顯示。
